### Build and Run

1. Clone or fork this repo;

Before you run your VM, you must change variablles in the `build.zig`

1. Install QEMU (and change `QEMU` path in the `build.zig`)
2. Install NASM (and change `NASM` path in the `build.zig`)

After those steps you're ready to run

1. Install Zig/Zls 0.16 (if you don't have it yet)

> [!TIP]
> For a first time, open the `build.zig` and find `runQemu` procedure.
> Replace my command (explicit QEMU path) with the your alias of QEMU 
> or your explicit `qemu-system-x86_64` path too.

Other details are placed in the repository or will be generated by Zig during the `zig build` steps.

2. `zig build`
3. `zig build run`

That's all. No more linker scripts or makefiles here.
